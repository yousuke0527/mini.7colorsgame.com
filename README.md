# Mini Games

AstroとTypeScriptで構築した、インストール不要で遊べるブラウザ向けミニゲーム集です。291本のゲームを日本語・英語のプレイ画面で提供し、検索、カテゴリ絞り込み、お気に入り、ローカルランキング、クリア記録に対応しています。

## 1. プロジェクト名

**Mini Games** — 短時間で遊べるCanvasベースのミニゲームを集めた静的Webサイト

`package.json`上のパッケージ名は `antigravity-mini-games` です。

## 2. 概要

ダウンロードやアカウント登録をせず、PCやスマートフォンのブラウザからすぐにゲームを選んで遊べる場を作ることが目的です。短い空き時間に遊びたい利用者に対し、多数のゲームを検索・分類して見つけやすくし、プレイ回数、お気に入り、ハイスコア、クリア状況を端末内に保存します。Astroによる静的ページ生成と、ゲーム単位のTypeScriptモジュール読み込みを組み合わせています。

## 8. 技術構成

ゲームと記事のメタデータはMarkdownで管理し、Astroが一覧・詳細ページを静的生成します。ゲーム画面を開くと、そのゲームIDに対応するTypeScriptモジュールを動的に読み込み、`canvas`上で描画と入力処理を開始します。お気に入りなどの利用履歴は外部サーバーへ送らず、利用者のブラウザ内に保存します。

```text
Antigravity_3/
├─ src/
│  ├─ components/        # ヘッダー、カード、広告などの共通UI
│  ├─ content/
│  │  ├─ games/          # 291本のゲームメタデータ
│  │  └─ blog/           # 26本の記事本文・メタデータ
│  ├─ games/             # ゲームごとのTypeScriptロジック
│  ├─ layouts/           # 共通HTML、SEO、スタイル
│  ├─ lib/               # 翻訳、カテゴリ分類、クリア条件
│  └─ pages/             # 日本語・英語・ブログ・サイトマップのルート
├─ public/
│  ├─ images/games/      # 291点のSVGサムネイル
│  └─ robots.txt         # クローラー向け設定
├─ astro.config.mjs      # Astro設定
├─ tailwind.config.mjs   # Tailwind CSS設定
├─ tsconfig.json         # TypeScript設定
├─ package.json          # 依存関係と実行コマンド
└─ package-lock.json     # 解決済み依存バージョン
```

処理の流れは「Markdown読込 → Astroによる静的HTML生成 → ブラウザで対象ゲームを動的読込 → Canvas描画 → 端末内へプレイ結果を保存」です。

### ゲーム追加の基本単位

既存READMEにあった拡張手順を、現在のスキーマに合わせて要約します。

1. `src/content/games/<game-id>.md`に`title`、`description`、`thumbnail`、`difficulty`、`category`、`publishDate`、`platform`を定義する。
2. `src/games/<game-id>/index.ts`で`controls`と`init(canvas)`をexportする。
3. `public/images/games/<game-id>.svg`にサムネイルを配置する。

この3点が同じゲームIDで結び付き、一覧、プレイ画面、サイトマップの生成対象になります。
