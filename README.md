# mini.7colorsgame.com
# ミニゲーム集プラットフォーム

本プロジェクトは、AstroとVanilla JS（Canvas API）で構築された、超軽量・高速ロード設計のブラウザミニゲーム集プラットフォームです。
AstroとTypeScriptで構築した、インストール不要で遊べるブラウザ向けミニゲーム集です。291本のゲームを日本語・英語のプレイ画面で提供し、検索、カテゴリ絞り込み、お気に入り、ローカルランキング、クリア記録に対応しています。

## ⚡️ パフォーマンス・設計の特徴
1. **ゼロJSによる超高速ロード (Lighthouse 95%+)**
   - ゲーム一覧や共通レイアウト部分のクライアントサイドJavaScriptロード量は **0KB** です。
2. **完全アセット遅延ロード (Code Splitting / Dynamic Import)**
   - 各ゲームのアセットやゲームロジックは、ユーザーがそのゲームのプレイ画面へ遷移した瞬間に初めてダウンロード（動的インポート）されます。
3. **極限まで軽量化された共通デザイン**
   - Tailwind CSSにより、使用されている最小限のスタイル定義のみを抽出し、CSSファイルサイズを極限まで小さくしています。

`package.json`上のパッケージ名は `antigravity-mini-games` です。

コアシステム（一覧画面や共通レイアウト）のコードを1行も変更することなく、以下の **3ステップ** だけで新しいゲームを完全に追加できます。
## 2. 概要

### Step 1: ゲームのメタデータ（Markdown）を定義する
ダウンロードやアカウント登録をせず、PCやスマートフォンのブラウザからすぐにゲームを選んで遊べる場を作ることが目的です。短い空き時間に遊びたい利用者に対し、多数のゲームを検索・分類して見つけやすくし、プレイ回数、お気に入り、ハイスコア、クリア状況を端末内に保存します。Astroによる静的ページ生成と、ゲーム単位のTypeScriptモジュール読み込みを組み合わせています。

`src/content/games/` ディレクトリの中に、新しく `[game-id].md` という名前のマークダウンファイルを作成します（例: `tetris.md`）。
